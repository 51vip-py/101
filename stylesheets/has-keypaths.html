Module: 101/has-keypaths

has-keypaths

require("101/has-keypaths")(obj, keypaths, deep) → {boolean|function}

Determines whether the keypaths exist and have the specified values. If obj is not provided findIndex will return a partial-function which accepts a obj as the first argument.

Parameters:
Name Type Argument Description
obj object <optional>

the object whose properties being checked
keypaths object | array

keypaths and values (object) or keypath strings expected to exist on the object (array)
deep boolean <optional>

deep equals when values are specified (objects are recursed), deep keypath existance (prototype) when only keys are specified
